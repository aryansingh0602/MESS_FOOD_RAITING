Statement
Problem Statement

Hostel students lack a simple way to record and track daily mess food quality.
Without proper logs, it's hard to give feedback or identify patterns in food quality.

Scope of the Project

A minimal Python CLI tool that lets users record meal ratings and store them in a CSV file.
It also allows viewing past ratings for basic tracking and analysis.

Target Users

Hostel students, mess committees, and anyone wanting an easy food rating log.

High-Level Features

Add ratings for breakfast, lunch, and dinner

Automatically save ratings in a CSV file

View all past entries

Fast and simple CLI interface

Problem-Solving Aspect

Provides a structured way to track food quality over time, helping students identify issues
and present data-backed feedback to mess management.

Future Enhancements

GUI version (Tkinter or Web)

Date-based filtering

Average rating calculations

Export to PDF

Admin dashboard for mess staff
