# Mess Food Rating System

A small command-line program to record and check ratings for mess food.  
I made this using basic Python and CSV storage.

## Features
- Add rating: name, dish, rating (1-5), comment
- Show overall average
- Show per-dish stats
- Export weekly summary to a text file

## How to run
python src/rating_app.py

## Files
src/ - main code  
data/ - ratings csv  
screenshots/ - sample runs  
report.docx - project report
